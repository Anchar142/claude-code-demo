# Claude Code 使用指南

## 一、快速开始
1.  安装 Claude Code CLI
    - macOS: `brew install claude-code`
    - Windows: 下载官方安装包
2.  配置环境变量
    - 填入 API Key（个人版/企业版）
    - 设置默认模型与上下文窗口大小
3.  连接项目，开始对话
    - `claude-code init` 初始化项目
    - `claude-code chat` 进入对话模式

## 二、常用指令速查（高频实用）
| 指令 | 作用 | 适用场景 |
|------|------|----------|
| `/compact` | 压缩历史对话，减少Token占用 | 多轮对话后上下文溢出 |
| `/cost` | 查看本次会话Token消耗与成本 | 控制开发预算 |
| `/status` | 查看会话状态、剩余额度、上下文大小 | 监控使用情况 |
| `/clear` | 清空当前会话历史 | 切换新任务时重置环境 |
| `/model [name]` | 切换模型（如 `claude-3-opus`） | 简单任务用轻量模型，复杂任务用高阶模型 |
